- 🛠️ Working across PHP/Laravel, React.js/Next.js, and TypeScript, with MySQL/PostgreSQL on the backend
- 🎯 Currently focused on go-to-market: marketplace launches (Dealify, AppSumo), pricing strategy, and a Laravel-based customer/admin panel (Blade + Tailwind + Alpine.js, Paddle billing)
- 🤝 Open to Senior Full-Stack / Product Engineer opportunities at SaaS companies building workflow automation tools
